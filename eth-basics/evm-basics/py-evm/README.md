# Py-EVM

## 目标

主要目标是通过Python实现为以太坊生态系统添砖加瓦：

* 支持以太坊1.0以及以太坊2.0/Serenity
* 文档齐备，有据可查
* 易于理解
* 具有清晰的APIs
* 运行速度快且节约资源
* 高度灵活，可支持：
  * 公链
  * 私链
  * 联盟链
  * 前沿研究
